# Guide d'utilisation — Urban Platform

Bienvenue dans le guide d'utilisation de l'interface d'administration d'Urban Platform.

Ce guide vous accompagne pas à pas dans la gestion du contenu du site : publications, événements, fiches pédagogiques, images, documents et paramètres.

## À qui s'adresse ce guide ?

Ce guide est destiné aux **éditeurs de contenu** et **administrateurs** de la plateforme. Aucune compétence technique n'est nécessaire.

## Accès à l'interface

L'interface d'administration est accessible à l'adresse : **`/admin`** de votre site.

> **Conseil :** Mettez cette adresse en favori dans votre navigateur pour y accéder rapidement.
