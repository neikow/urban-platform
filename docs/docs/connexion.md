# Se connecter

## Accéder à la page de connexion

Rendez-vous sur la page de connexion en ajoutant `/admin` à l'adresse du site.

## Remplir vos identifiants

1. Saisissez votre **adresse e-mail** dans le premier champ.
2. Saisissez votre **mot de passe** dans le second champ.
3. Cochez **"Se souvenir de moi"** si vous souhaitez rester connecté.
4. Cliquez sur le bouton **"S'identifier"**.

## En cas de problème de connexion

Si vous ne parvenez pas à vous connecter :

- Vérifiez que votre adresse e-mail est correctement saisie (sans espace en début ou fin).
- Vérifiez que le verrouillage des majuscules n'est pas activé.
- Contactez votre administrateur pour réinitialiser votre mot de passe.

## Se déconnecter

Pour vous déconnecter, cliquez sur votre **adresse e-mail** en bas à gauche de la barre latérale, puis sélectionnez **"Se déconnecter"**.
